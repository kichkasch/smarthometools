smart home tools
================

Tools and scripts for integration of smart home devices such as broat band routers 
(mainly AVM Fritzbox) and Smart Home Management Environments (mainly Loxone)

* Carddav2FritzboxXML: Create XML from Carddav server to be manually imported as phone book in AVM Fritzbox
* calllist2mysql: Downloads call log from fritzbox and pushes details to mysql database
* fritz_missedcalls: Downloads call log from fritzbox (via UPNP) and triggers notification in smart home
environment for missed calls
* loxFetchWetter: Fetches weather information from wonderworld and pushes it to Loxone Smart home server
* loxFhem: documentation on integration FHEM and Loxone
* magichome. PicoC program to be used in Loxone to control Magic Home LED controller
* rainifritz: small tool, which allows for manual importing of thunderbird contacts into AVM Fritzbox
